Lightning Detection System

Keys On Kites

Product Description

The Final Replacement to A Nonideal Kite Lightning Influx Node (FRANKLIN) is a system of nodes setup to detect, locate, and triangulate lightning strikes. This allows the scientific community to further their understanding of how lightning works and furthers application in forest fire prevention.

Each node in our system consists of many different parts. When the lightning strikes, the voltage is detected by an aerial antenna into the signal conditioning board. The signal is then transferred to to the Red Pitaya and converted from analog to digital. If a certain voltage threshold is passed, then a sampling period is commenced. The timestamp of the voltage threshold pass allows our set of nodes to triangulate a strike.
